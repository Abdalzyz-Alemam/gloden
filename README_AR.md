# دليل توثيق منيو كافيه قولدن (Golden Cafe)

هذا الملف يشرح هيكلية المشروع وكيفية التعامل معه وإجراء التعديلات البرمجية عليه.

## 1. هيكلية الملفات (File Structure)

*   **`index.html`**: الهيكل الأساسي للموقع. يحتوي على رأس الصفحة (Header)، أزرار التصنيفات (Categories)، حاوية البحث (Search)، وحاوية عرض المنتجات (Menu Grid).
*   **`src/main.js`**: المحرك الأساسي للموقع. يحتوي على بيانات المنتجات (menuItems) وكافة الدوال البرمجية (البحث، تصفية التصنيفات، النوافذ المنبثقة، وتغيير اللغة).
*   **`src/index.css`**: يحتوي على التنسيقات العامة واستدعاء مكتبة Tailwind CSS.
*   **`package.json`**: ملف إدارة المكتبات والاعتماديات.

---

## 2. كيفية إضافة أو تعديل أصناف الطعام (Menu Items)

يتم تخزين كافة الأصناف في مصفوفة `menuItems` داخل ملف `src/main.js`. 

### هيكل الصنف الواحد:
```javascript
{ 
  id: 1, 
  title: 'اسم الصنف (الحجم)', 
  price: 'السعر', 
  category: 'التصنيف', 
  image: 'رابط الصورة', 
  description: 'وصف المنتج (اختياري)' 
}
```

### ملاحظات هامة عند الإضافة:
1.  **الأحجام**: إذا كان للصنف أحجام مختلفة (صغير، وسط، كبير)، يجب إضافة كل حجم كصنف مستقل. النظام سيقوم تلقائياً بتجميعهم في بطاقة واحدة بناءً على تشابه الاسم قبل القوسين.
    *   *مثال*: `بيتزا (صغير)` و `بيتزا (كبير)` سيظهران في بطاقة واحدة باسم `بيتزا` وبداخلها السعرين.
2.  **التصنيف (category)**: يجب أن يطابق تماماً قيمة `data-category` الموجودة في أزرار التصنيفات في ملف `index.html`.

---

## 3. كيفية إضافة تصنيف جديد (New Category)

لإضافة قسم جديد (مثلاً: "باستا")، اتبع الخطوات التالية:

### أولاً: في ملف `index.html`:
أضف زراً جديداً داخل حاوية التصنيفات:
```html
<button data-category="pasta" class="category-btn ...">
  <i data-lucide="utensils" class="..."></i>
  <span data-ar="باستا" data-en="Pasta">باستا</span>
</button>
```

### ثانياً: في ملف `src/main.js`:
أضف الأصناف الجديدة واجعل قيمة `category` هي `pasta`:
```javascript
{ id: 200, title: 'باستا خضار', price: '5,000', category: 'pasta', ... }
```

## 4. نظام اللغات (Multilingual Support)

الموقع يدعم اللغتين العربية والإنجليزية. يتم ذلك عبر سمات (Attributes) مخصصة في HTML:
*   `data-ar`: النص الذي سيظهر عند اختيار اللغة العربية.
*   `data-en`: النص الذي سيظهر عند اختيار اللغة الإنجليزية.

يتم التبديل برمجياً عبر دالة `langToggle` في `main.js` التي تقوم بتغيير اتجاه الصفحة (RTL/LTR) وتبديل النصوص.

---

## 5. ميزات برمجية مدمجة

*   **البحث (Search)**: يبحث في الأسماء والأوصاف بشكل لحظي.
*   **تجميع الأسعار (Price Grouping)**: دالة `renderItemsFromList` تقوم تلقائياً بفرز وتجميع المنتجات ذات الأحجام المختلفة لتوفير مساحة العرض.
*   **التجاوب (Responsiveness)**: الموقع مصمم ليعمل بشكل مثالي على الهواتف (Mobile-First) مع دعم كامل للشاشات الكبيرة.
*   **الأيقونات**: نستخدم مكتبة **Lucide** للأيقونات، ويمكن تغيير أي أيقونة بتعديل قيمة `data-lucide`.

## 6. نصائح للتطوير

*   للحفاظ على سرعة التحميل، يفضل استخدام صور بحجم صغير أو روابط مضغوطة.
*   عند تعديل أي كود برمي، تأكد من تشغيل `npm run build` للتأكد من سلامة البناء.
*   يمكن تغيير الألوان بسهولة عبر تعديل قيم `golden` و `black-gold` في تنسيقات CSS داخل `index.html`.
